# Warning
This "browser" does not support the following:
```
WebGL / WebGL2
WebGPU
Extensions
Add-ons
Widevine DRM
Encrypted Media Extensions (EME) (May exist)
Proxy APIs'
Service Workers (Sometimes works, may not)
Push Notifications (Often blockedd)
Platform Authenticators (Windows Hello or Touch ID)
WebAssembly (WASM) (Large projects fail)
SharedArrayBuffer (Some games and embeds can fail)
Periodic Background Sync (Websites may pause)
Google Account integrations (For obvious reasons)
User Agent Detection (Some sites may misidentify you, good if you don't want to be fully exposed)
```

# Requirements
Node JS 10 or higher.

# How to setup
Install the needed Node imports using the following command:
```
npm install
```
Download the files and put the folder inside of your C: directory.
Run this command in your terminal:
```
npm run
```
The app should launch as normal.
Alternaitvely, you can move the shortcut provided to your desktop or elsewhere to start the browser.
